美佳のタイプトレーナ Ｖｅｒ２．０６

　このタイプ練習ソフトは、初めてキーボードにさわる初心者から、
一分２００ー４００文字の高速のタイプ速度を目指すプロのオペレータまで、
広い範囲での練習が可能なタイプト練習ソフトです。

英文字、ローマ字の練習が可能です。
Ｃ言語や、ＭＳＤＯＳコマンドなど幅広い練習メニューを備えています。

Windows 95、98、NT、ME、2000、XP で作動します。

従来リリースされたDOS汎用の美佳のタイプトレーナシリーズも
Windows 95、98 のDOS窓で作動します。

美佳シリーズのバージョン番号 Ver1.XXはDOS汎用、
Ver2.XXはWindow95、98、NT、ME、2000、XP用です。
Windows 3.10用にはリリースの予定はありませので、DOS汎用版を使って下さい。

美佳のタイプトレーナシリーズは、教育用のタイプ練習ソフトです。
大学、短大、専門学校、高校、中学校、小学校などの教育機関で広く使われています。
すべてフリーソフトですので、教材として自由にコピーしてお使い下さい。
プログラム著作権は，今村二朗が保有します。
非営利目的のネットへは、自由に転載してかまいません。
転載を行ったあとに通知を下さい。
商用目的のネットへの転載は事前に許可を受けて下さい。
ただしダイアルＱ２を使用したネットへの転載はお断りします。
